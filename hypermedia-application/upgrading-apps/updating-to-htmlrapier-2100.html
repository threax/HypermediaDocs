Updating to HtmlRapier 21.0.1

This will describe how to update to HtmlRapier 21.0.1. This version does not make major changes, but it gets rid of the polyfill.js file and this needs to be adapted.

Update package.json

First update package.json to the latest HtmlRapier 21 version. At this time this is 21.0.1.

"htmlrapier": "^21.0.1",

Next run the install npm task to update the installed library.

Update bundleconfig.json

Next you need to update bundleconfig.json. Here remove the following line:

"wwwroot/lib/polyfill.js",

This should complete the upgrade. HtmlRapier will now build and work with ES2015, but you won't be able to minify it unless you use a minifier compatible with ES2015. As a result just leave it on ES5 for now and keep using BundlerMinifierCore. Looking into alternatives like terser or just using the browser module capabilities.
