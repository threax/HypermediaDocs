Updating to HtmlRapier 20

HtmlRapier 20 removes the dependency on @types/es6-promise. This will cleanup the code and dependencies since that package has long been deprecated. The major version number changed because a few configuration changes need to be made to apps.

First open package.json and update your htmlrapier reference to 20.0.2.

Also in that file remove the following package if you have it under devDependencies: @types/node. Run npm install to install these changes.

Next add the following to your tsconfig.json under compilerOptions:

    "lib": [
      "dom",
      "es5",
      "es2015.promise"
    ],

That will instruct typescript to use the built in promise definitions from the es2015 library.
