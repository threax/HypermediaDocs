Threax Hypermedia Framework and Architecture

Updating to Net 6.0

Updating to Net 6.0 is mostly just updating nuget packages, but a couple of things have been removed.

Threax.SqLite

The upgrade function to update databases to the netcore3.1 guids has been removed.

Update .net version

Update all

  <TargetFramework>net5.0</TargetFramework>

to

  <TargetFramework>net6.0</TargetFramework>

Typescript Compilation

Visual Studio 2022 has a new typescript compilation method. Allow it to be enabled when prompted then add.

    <TypeScriptCompileBlocked>true</TypeScriptCompileBlocked>

to the csproj. For now there seems to be an issue with how this compiles vs how htmlrapier is setup.
