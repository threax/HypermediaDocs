Mass Library Update for NetCoreApp3.1

This update will remove all the netstandard2.0 versions of the Asp.Net Core libraries. This will remove the annoying warning when building about things being loaded more than once. For the most part just update all the Threax libraries. They should go up one version. They are all updated, so I'm not going to list them out.

In addition to this also add a direct reference to your csproj for the following library:

<PackageReference Include="IdentityServer4.AccessTokenValidation" Version="3.0.1" />

This will make sure the IdentityServer4.AccessTokenValidation package is kept up to date with the rest of the top level packages.

There should be not breaking changes in the apps. But a huge number of older libraries should be uninstalled when you update.
