Updating to .Net Core 3.1

The .net core 3.1 update is minor compared to most of them. All libraries work under 3.0 and 3.1, so its a version update.

The official update docs are here.

Retarget to 3.1

Update the csproj change the target framework to netcoreapp3.1.

<TargetFramework>netcoreapp3.1</TargetFramework>

Update Bundler Minifer Core

Also in the csproj update your BundlerMinifierCore version.

<DotNetCliToolReference Include="BundlerMinifier.Core" Version="3.2.435" />

Update Nuget Packages

Update any Microsoft .Net Core libraries to 3.1 or above. None of the Threax libraries require any special updates to work with 3.1.

Update Dockerfiles

Update your Dockerfile to use .Net Core 3.1 and the threax/build image for 3.1.

FROM mcr.microsoft.com/dotnet/core/aspnet:3.1 AS base

and

FROM threax/build-dotnet:3.1.0 AS build

This should complete the upgrade to .Net Core 3.1.
