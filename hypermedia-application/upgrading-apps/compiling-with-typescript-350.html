Compiling With Typescript 3.5.0

Apps may have problems compiling with Typescript 3.5.0 and above. You will get an error similar to error TS2304: Cannot find name 'Iterable'. This is because Typescript is no longer able to find the iterable library by default.

To fix this add "es2015.iterable" to the "lib" array in your tsconfig.json file. When you are done it should look something like the following:

"lib": [
  "dom",
  "es5",
  "es2015.promise",
  "es2015.iterable"
],

The Typescript build should now work correctly.
