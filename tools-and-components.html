Tools and Components

These are the primay tools and components we use to build apps.

  • Asp.net Core - Primary framework
  • Visual Studio 2019 - IDE
  • Nuget - This is the package manager for .net.
  • Identity Server 4 - Open ID Connect Middleware, will build an Identity Server using this.
  • Node.js - Javascript runtime, used to process and compile the front end code for our applications.
  • Npm - The Node Package Mangager. This program will retrieve the front end packages.
  • Typescript - Really nice javascript syntax sugar, integrates well with Visual Studio including code analysis and refactoring, also allows you to use modern javascript techniques and compile them down to ECMAScript 5 (or other targets).
  • HtmlRapier - Lightweight javascript front end library.
  • jsns - Javascript module manager that allows for async loading (out of order) and provides wrappers for wrapping up node modules inside files. Also enables you to compile modules to one or many files instead of having to bundle it all up.
  • Boostrap - Css and UI framework.
  • Threax Hypermedia Framework - Extensions to .net core to enable HATEOAS (Hypermedia as the Engine of Application State) apis and identity management for microservices/microapps using identity server.
  • SqLite is the default database provider, but any database back end for Entity Framework Core should work. All database access in the templates and core services is done through the ef abstractions.

Apps

In addition to the above tools each environment will have a few apps that are needed to make everything work. This is a microservice framework, so the following microservices will be available in any environment.

Identity Server - This service provides identity management for apps. The reference implementation in Threax.IdServer handles human user accounts and the applications defined in it as users. This allows you to give permissions to individual applications to access other applications.

  • Local Dev Environment - https://localhost:44390

App Dashboard - This app allows you to edit the apps and properties of the Identity Server.

  • Local Dev Environment - https://localhost:44391
